Newon 사이트 업로드용 묶음
============================

이 폴더 안의 모든 파일·폴더를 웹 서버 문서 루트에 그대로 올리세요.
(FileZilla 등으로 publish 폴더 내용 통째 업로드, 또는 체크리스트 보고 하나씩)

필수 구조:
  index.html
  styles.css
  ox-month.css
  assets/        (logo.png + ox-month/*.png)
  CNAME          ← GitHub Pages만 (다른 호스는 생략 가능)

GitHub에는 ZIP이 아니라 git push 로 올리세요.

소스 수정 후 다시 만들기:
  ./scripts/prepare-publish.sh
