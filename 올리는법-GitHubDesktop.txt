GitHub Desktop 으로 올리기 (그림 없이 단계만)
============================================

1. desktop.github.com → 설치 → 로그인 (newonapp 계정)

2. File → Clone repository
   • URL: https://github.com/newonapp/Newon-site.git
   • Local path: 원하는 폴더 (예: Documents/Newon-site)

3. 맥에서 zip 준비
   • 터미널: cd /Users/kyungnawon/Newon && npm run 묶기
   • 또는  업로드-준비.command  더블클릭

4. Newon-site-upload.zip 더블클릭 → 압축 풀기

5. 풀린 폴더 안의 「모든 것」을 Clone 한 폴더로 드래그
   (ko, en, scripts, index.html … 전부. 덮어쓰기 OK)

6. GitHub Desktop 왼쪽에 변경 파일 목록이 보임
   • Summary: Site update
   • Commit to main
   • Push origin (또는 상단 Push)

7. 브라우저에서 확인
   • github.com/newonapp/Newon-site → Actions → 성공 여부
   • https://www.newon.app

주의: zip 파일 자체를 GitHub에 올리지 말고, 「압축 푼 내용」을 올리세요.
