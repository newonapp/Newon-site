Noting – Löschen Sie Ihr Konto

App: Noting · Entwickler: Newon

Sie können Ihr Konto direkt in der App löschen.

So löschen Sie Ihr Konto

  1. Öffnen Sie die App
  2. Gehen Sie zu Einstellungen
  3. Wählen Sie Konto aus
  4. Tippen Sie darauf Konto löschen

Beim Löschen

  • Alle Benutzerdaten werden sofort gelöscht.
  • Gelöschte Daten können nicht gelöscht werden wiederhergestellt.
