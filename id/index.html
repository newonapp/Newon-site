OX MONTH adalah

aplikasi untuk menandai hari tanpa log berat dan mengelola kebiasaan serta target bulanan.

OX MONTH cocok jika Anda ingin mencatat hari tanpa beban input berlebih. Aplikasi ini mendukung 13 bahasa—Korea, Inggris, Jepang, Spanyol, Portugis (Brasil), Prancis, Jerman, Hindi, Indonesia, dan lainnya—dan tersedia di 173 negara.
