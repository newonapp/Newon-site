PiggyUp — Supprimez votre compte

Application : PiggyUp · Développeur : Newon

Vous pouvez supprimer votre compte directement dans l'application.

Comment supprimer votre compte

  1. Ouvrez l'application
  2. Allez dans Paramètres
  3. Sélectionner un compte
  4. Appuyez Supprimer le compte

Lorsque vous supprimez

  • Toutes les données utilisateur sont immédiatement supprimées.
  • Les données supprimées ne peuvent pas être récupéré.
